Oque é:
O codigo Secretaria Semi se trata de um sistema de secretaria escolar, aonde você como administrador, professor ou até mesmo como responsavel consege ter controle sobre o progresso de seu filho na escola,
nele você consegue controlar questão de nota, penalidades etc ( para administradores), como também a reserva de salas, publicação de notas etc para professores, para responsaveis você conseguem ter a visualização
de todas as informações de comportamento de seu filho, seja nota, se ele recebeu alguma penalidade (ex: suspensão) ou até mesmo como está seu relatorio de faltas

Como iniciar:
Para se iniciar o codigo primiro se deve criar um superuser no django admin, após isso acesse o painel de admin e configure uma conta para a secretaria, logo em seguida entre no site com as informações de login cadastradas,
com isso você irá ter acesso como se fosse uma secretaria, aonde voce ira conseguir realizar diversas ações sendo de publicar uma nota até mesmo adicionar um professor ao sistema da escola
